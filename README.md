## 📌 Project Overview

The goal of this project was to design a modern and visually appealing coffee shop interface that:

* Provides a smooth and enjoyable browsing experience
* Highlights products
* Maintains a clean and minimal design style
* Reflects a strong and consistent brand identity

---

## 🎯 Problem

Many coffee shop websites suffer from:

* Cluttered layouts
* Poor product visibility
* Weak visual hierarchy
* Lack of engaging user experience

---

## 💡 Solution

Aroma Brew solves these problems by:

* Using a clean and structured layout
* Applying strong typography and spacing
* Creating visually rich product cards
* Designing clear navigation and sections

---

## 🎨 Design Highlights

* Modern & minimal UI
* Warm coffee-inspired color palette ☕
* Clear visual hierarchy
* Card-based product layout
* Smooth and simple navigation

---

## 🧠 UX Thinking

This project focuses on:

* Easy navigation between categories (Coffee / Non-Coffee / Desserts)
* Clear call-to-actions (CTA)
* Reducing user effort
* Creating a طبيعي flow
