# ☕ Aroma Brew - Coffee Shop UI/UX Design

Aroma Brew is a modern UI/UX design concept for a coffee shop, created to deliver a smooth, visually appealing, and user-friendly digital experience.

This project focuses on combining clean aesthetics with functional design to reflect the cozy and premium atmosphere of a café brand.

---

## 🎯 Project Overview

The goal of this project was to design a coffee shop experience that:
- Feels modern and minimal
- Highlights products  
- Improves user interaction and navigation
- Reflects a strong brand identity

---

## 🎨 Design Highlights

- Clean & modern UI
- Strong visual hierarchy
- Consistent color palette inspired by coffee tones
- Simple and intuitive layout
- Engaging product sections (coffee, drinks, offers)

---

## 🧠 UX Thinking

- Focus on user comfort and easy navigation
- Clear call-to-actions (CTA)
- Organized content flow
- Designed to reduce friction

---

## 🛠 Tools Used

- Figma (UI Design & Prototyping)
